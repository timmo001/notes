The `notes` CLI is the direct interface to the same note service exposed through MCP.

```bash
notes list
notes read --path <path>
notes write --path <path> --stdin
notes handoffs
notes mcp
```

Use `notes help` or `notes <command> --help` for terminal help. The [command reference](/cli/commands/) is generated from the same CLI registry.

Shell completions are generated by [`notes completions`](/cli/completions/).
